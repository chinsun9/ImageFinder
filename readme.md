# ImageFinder

- 기존에 안드로이드로 만들어진 [ImageFinder](https://github.com/ApexCaptain/ImageFinder)프로젝트를 참고하여 ios 앱으로 만들어 보았습니다.
- 카카오 이미지 검색 api를 이용해서 만들었습니다.
